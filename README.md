# Cloud Security Monitoring Lab (AWS)
This project demonstrates a cloud security monitoring environment built in AWS to detect suspicious activity using centralized logging, monitoring, and automated alerts.

## Objectives
The goal of this lab is to simulate a cloud environment and implement security monitoring controls to detect potential threats.

Key Objectives include:
- Implement AWS CloudTrail for API activity logging
- Monitoring logs using Amazon CloudWatch
- Detecting suspicious behavior such as failed logins and privilege escalation
- Generating automated alerts using Amazon SNS
- Using AWS GuardDuty for intelligent threat detection
